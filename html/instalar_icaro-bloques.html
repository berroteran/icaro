1.5. Instalación manual de icaro-bloques.

Para instalar manualmente icaro-bloques hay que seguir los siguientes pasos (como root):
  1. #> yum -y install pygame pywebkitgtk pygtksourceview
  2. #> yum -y install sdcc
  3. copiar el archivo pic18f4550.h en la carpeta include a /usr/share/sdcc/include/pic16/
  4. copiar el archivo pic18f2455.h en la carpeta include a /usr/share/sdcc/include/pic16/
  5. copiar el archivo libdev18f4550.lib en la carpeta include a /usr/share/sdcc/lib/pic16/
  6. Dar permiso de ejecución al binario docker en la carpeta tools/bin/.
